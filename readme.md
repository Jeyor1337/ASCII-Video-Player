# ASCII Video Player

This project allows you to convert video files into ASCII art and play them in your terminal.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd asciivideo
    ```

2.  **Create a virtual environment:**
    ```bash
    python -m venv venv
    source venv/bin/activate
    ```

3.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

## Usage

The project is divided into two main components: the `translator` and the `interpreter`.

### 1. Translate a Video

The `translator.py` script converts a video file (e.g., MP4) into a custom `.obj` file containing the ASCII art representation of the video.

```bash
python translator.py --input /path/to/your/video.mp4 --output /path/to/your/output.obj
```

**Arguments:**

*   `--input`: The path to the input video file.
*   `--output`: The path where the output `.obj` file will be saved.
*   `--width`: (Optional) The desired width of the ASCII art output. Defaults to 120 characters.

### 2. Play the ASCII Video

The `interpreter.py` script plays the `.obj` file generated by the translator in your terminal.

```bash
python interpreter.py --input /path/to/your/output.obj
```

**Arguments:**

*   `--input`: The path to the input `.obj` file.
